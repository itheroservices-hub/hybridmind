## 🚀 Quick Start (3 Steps):

### 1️⃣ Get Your OpenRouter API Key
```
Go to: https://openrouter.ai/keys
Sign up (free) and create an API key
Copy it (starts with sk-or-v1-...)
```

### 2️⃣ Set the API Key in PowerShell
```powershell
$env:OPENROUTER_API_KEY="sk-or-v1-your-actual-key-here"
```

### 3️⃣ Run a Demo
```powershell
# Basic demo - see how agents work
py -3.13 autogen_basic_demo.py

# Advanced demo - agents collaborate on HybridMind tasks
py -3.13 autogen_hybridmind_demo.py
```
